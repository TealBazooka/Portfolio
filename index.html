Child's Play Charity - Used Item Donation

This school project started as a redesign of the Child's Play Charity website, but because they were already redesigning it, I opted to design a feature using their new redesign as a platform. Through talking with Child's Play and a competitive analysis I settled on developing a user flow for used item donations via local donation centers or by mail using shipping assistant modal.
